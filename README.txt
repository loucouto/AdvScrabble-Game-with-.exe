---------------------------------------INSTRUCTIVE WRITTEN IN ENGLISH------------------------------------------------------------------------

To use the following proyect, you have to take account of doing the following steps:
1- To open the project, open the folder:
a- AdvScrabbleWin
b- Double click in AdvScrabble.exe
c- Press the buttton "Play".

In other words, go to the directory: ...\AdvScrabbleWin\AdvScrabble.exe and press the button "Play". Clarification: the three suspended points are refering
to the current folder place.

2- As this is an experimental project and this has'nt been finished yet, it could have some bugs.

3- This game have resolution graphics issues because there are differences between the programming view and the view after building the game.

4- This game have only a level made, then if you pass the first level and you want to try it again, do the following steps:
a- If the game is running, stop it.
b- Go to ...\AdvScrabbleWin\AdvScrabble_Data\StreamingAssets\RegistroMapa.json. If there is something written, clean, save and close it.
c- Idem for ...\AdvScrabbleWin\AdvScrabble_Data\StreamingAssets\RegistroPantalla.json.
d- Run the game again.

5- If the life's count is equal to cero and you don't want to wait 20 minutes to have the first life, do the following steps;
a- If the game is running, stop it.
b- Go to  ...\AdvScrabbleWin\AdvScrabble_Data\StreamingAssets\Corazon.json.
c- Update the value "count_lifes":0. Instead, put "count_lifes":5. Then save and close it.
d- Go to  ...\AdvScrabbleWin\AdvScrabble_Data\StreamingAssets\ProxVida.json. If there is something written, clean, save and close it.
e- Run the game again. 

6- The second bonus (exchange), is disabled for this level (level 1)

7- If you want to use some of the remaining bonus and its value is equal to cero, do the following steps:
a- If the game is running, stop it.
b- Go to ...\AdvScrabbleWin\AdvScrabble_Data\StreamingAssets\Bonus.json.
c- Update all the values that is written "count_bonus":0. Instead, put "count_bonus":3. Then save and close it.
e- Run the game again.

Thanks for prove the game.
